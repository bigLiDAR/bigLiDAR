bigLiDAR

Big LiDAR data needs a strong data base, spatial indexing and scalability. bigLiDAR provides an open solution hosted on GitHub, built on MongoDB.

More

What It Is

bigLiDAR provides software tools and defines a framework for storing point cloud files in a NoSQL database. bigLiDAR is based on the proven and scalable architecture of MongoDB. Point clouds of arbitrary size can be stored in a database, queried by metadata or spatial location and retrieved to the local filesystem.

Who It's For

Organization or individuals who hold large collection of LiDAR files face the challenge of systematic archival of point clouds. bigLiDAR bring a low-cost, scalable and open source solution to this challenge.
